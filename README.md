# Proyecto de Página Web

## 📌 Descripción
Este repositorio contiene una práctica de desarrollo de una página web básica, creada con el objetivo de aprender y mejorar habilidades en **HTML, CSS y JavaScript**.

## 🤖 Herramientas utilizadas
Durante el desarrollo de este proyecto se ha utilizado **ChatGPT (GPT-5.3-mini)** como herramienta de apoyo para la generación de código, resolución de dudas y mejora del proceso de aprendizaje.

## 🚀 Objetivo del proyecto
- Practicar la creación de estructuras web básicas
- Mejorar habilidades de diseño y maquetación
- Aprender buenas prácticas de desarrollo web y gestión LLMS
